node.js
node js itu adalah js runtime yang di buat didalm js v8/untuk membuat aplikasi asycronus
node.js itu bukan bahasa pemrogramana tetapi itu runtime untuk menjalankan js, 
runtime itu tempat kita menjalankan program
node bisa mengakses memory,IO,network,file system,dll
node js dibuat karena js dengan teknologi v8 hanya bisa di jalankan di browser, dan akhirnya dengan adanya node js maka js&v8 ini bisa keluar dari browser dan bisa di gunakan, penyiptanya iala ryan dahl

1. asynchronus & Event driven
2. non-blocking
3. cross platfrom
4. sinfle threaded but highly scalable
5. MTI License
6. NPM

    "test": "echo \"Error: no test specified\" && exit 1"
  }", test itu secara default kita bisa memangil scriptnya dengan keyword test

dan ketika di ubah semisal awalnya adalah = ("scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  }") menjadi => (   "scripts": {
    "start" : "node app",
    "dev" : "ls",

  });
maka ketika nama ada yang bukan standart seperti "dev" maka jalankannya dengan model "npm run dev"
